# 운영체제

*OS, Operating System*

- 컴퓨터 하드웨어를 관리하고, 다른 프로그램이 CPU·메모리·저장장치·입출력 장치 등을 사용할 수 있도록 중간에서 연결·조율하는 **시스템 소프트웨어**입니다.
- Windows, macOS, Linux 등이 대표적입니다.

---

# 프로세스

*Process, 프로세스*

- **운영체제**가 관리하는 **실행 중인 프로그램의 단위**입니다. 코드·데이터·자원을 두고, 그 안에 하나 이상의 **스레드**를 둘 수 있습니다.

---

# 메모리

*Memory, RAM, 주기억장치*

- 프로그램 코드와 실행 중인 데이터가 올라가 **CPU가 읽고 쓸 수 있는 저장 공간**입니다.

---

# 오픈소스

*Open Source, 오픈 소스*

- **소스 코드가 공개**되어 누구나 열람·수정·재배포할 수 있도록 허용하는 소프트웨어·라이선스 모델입니다.

---

# 클래스 로더

*Class Loader, 클래스로더*

- JVM이 실행에 필요한 **`.class` 바이트코드를 찾아 읽고**, 메서드 영역 등 **메모리에 적재(로드)**하는 JVM 하위 시스템입니다.

---

# 메서드

*Method, 메소드*

- 클래스 또는 인터페이스 안에 선언된 **동작의 한 단위**입니다.
- 이름을 붙여 호출하면 그 안의 문장이 실행됩니다.

---

# 인터프리터

*Interpreter, 해석기*

- 바이트코드 명령을 **한 단위씩 순서대로 읽어 곧바로 실행**하는 방식입니다.

---

# JIT 컴파일러

*JIT, Just-In-Time Compiler, 제때 컴파일러*

- 실행 중에 **자주 호출되는 코드(핫스팟)**를 감지해 바이트코드를 **네이티브 기계어로 번역·캐시**하는 최적화 구성 요소입니다.

---

# 가비지 컬렉터

*GC, Garbage Collector, 가비지 컬렉션*

- 더 이상 도달할 수 없는(참조되지 않는) 객체가 차지한 **힙 메모리를 자동으로 회수**하는 JVM 메커니즘입니다.

---

# JVM

*Java Virtual Machine, 자바 가상 머신*

- **바이트코드(`.class`)를 실행하는 자바 런타임**입니다.
- OS 위에서 클래스 로더·실행 엔진·가비지 컬렉터 등으로 동작하며, **같은 바이트코드를 JVM이 설치된 환경에서 실행**할 수 있게 해 플랫폼 독립성을 제공합니다.

---

# 스레드

*Thread, 쓰레드*

- 프로세스 안에서 **실행 흐름 하나**를 담당하는 단위입니다. 스레드를 여러 개 두면 **여러 코드 경로가 동시에(또는 CPU 시간을 나눠) 진행**될 수 있습니다.

---

# 프로그램 카운터 레지스터

*Program Counter Register, PC Register*

- 스레드마다 하나씩 있으며, **현재 실행할(또는 실행 중인) 바이트코드 명령의 주소**를 가리킵니다.
- 실행 중인 메서드가 `native`인 경우에는 규격상 PC 값이 정해지지 않을 수 있습니다.

---

# JNI

*JNI, Java Native Interface, 자바 네이티브 인터페이스*

- 자바 코드와 **C/C++ 등 네이티브 코드** 사이에서 메서드를 호출하고 값을 주고받기 위한 **규약·API 묶음**입니다.
- `native` 메서드를 선언하고 네이티브 라이브러리(`.dll`, `.so` 등)와 연결할 때 씁니다.

---

# 네이티브 메서드 스택

*Native Method Stack*

- `native` 메서드처럼 **JVM 바이트코드가 아닌 네이티브 코드**(JNI 등)가 실행될 때 쓰는 **호출 스택**입니다.
- 스레드마다 두며, 자바 메서드가 쓰는 **자바 가상 머신 스택**과는 **별도의 영역**입니다.

---

# JDK

*Java Development Kit, 자바 개발 키트*

- 자바 **개발·빌드에 쓰는 도구**(컴파일러 `javac`, 디버거, 문서·패키징 도구 등)와, 프로그램 **실행에 필요한 런타임**(JVM·표준 클래스 라이브러리)을 **함께 담은 배포물**입니다.

---

# JRE

*Java Runtime Environment, 자바 실행 환경*

- 자바 **애플리케이션 실행**에 필요한 환경으로, **JVM**과 실행 시 참조하는 **표준 클래스 라이브러리**로 이루어집니다.

---
